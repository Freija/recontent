Content Recommendation API

This Content Recommendation API was developed by Freija Descamps, Kuan Ying Fang and J. L. during the 2016 CDIPS Data Science workshop. Frank Cleary was the mentor for the team. The goal of the project was to design a recommendation API framework that can handle multiple corpora and uses different recommendation engines.

Try it out!

API Usage

The application has one endpoints:

GET /api/recommend/v1.0/<corpusname>

Provide a corpus_name as part of the GET request. Currently the options are: speech, simple-wiki and arx1501.

Responds with a list containing 5 recommendations. It supplies a redirect URL, score and title for each one

Example:

$ curl 'https://recommender.website/api/recommend/v1.0/speech' -X GET -d 'url=http://www.people.com/article/melania-trump-michelle-obama-similar-convention-speeches'
